# CWP 壁纸定时切换 — 需求开发说明书

## 1. 项目概述

CWP（壁纸定时切换）是一款 Android 原生应用，允许用户按预设计划自动更换设备桌面壁纸和锁屏壁纸。

- **包名**：`com.CWP.app`
- **最低 SDK**：24 (Android 7.0)
- **目标 SDK**：34 (Android 14)
- **语言**：Kotlin
- **UI 框架**：Material Components + AppCompat
- **架构**：单 Activity（`MainActivity`）+ BroadcastReceiver

---

## 2. 功能需求

### 2.1 定时任务管理

| 需求 | 描述 |
|------|------|
| 创建任务 | 用户选择日子类型 → 设定时间 → 选择图片 → 选择壁纸类型，完成创建 |
| 删除任务 | 点击卡片垃圾桶图标，确认后删除，同时清除对应闹钟 |
| 任务列表 | 按时间升序排列，显示缩略图、时间、日子类型、文件名、壁纸类型 |
| 持久化 | 全部任务以 JSON 格式存储在 `SharedPreferences` 中 |

### 2.2 日子类型

| 类型 | key | 说明 |
|------|-----|------|
| 仅一次 | ONCE | 执行后自动删除任务 |
| 每天 | EVERYDAY | 每日触发 |
| 工作日 | WEEKDAY | 周一至周五 |
| 周末 | WEEKEND | 周六、周日 |
| 上班日 | WORKDAY | 工作日且非法定节假日 |
| 节假日 | HOLIDAY | 仅法定节假日（内置 2024–2026） |

### 2.3 壁纸类型

- **桌面壁纸**：`FLAG_SYSTEM`
- **锁屏壁纸**：`FLAG_LOCK`
- **两者同时**：`FLAG_SYSTEM | FLAG_LOCK`

### 2.4 设置功能

- **夜间模式**：开启 / 跟随系统（互斥 Switch）
- **主题色**：9 色可选（红、橙、黄、绿、青、蓝、紫、黑、白），全局应用至工具栏、FAB、状态栏
- **隐藏最近任务**：从系统最近任务列表中隐藏应用

### 2.5 系统集成

- **AlarmManager 精确闹钟**：使用 `setAlarmClock()` 确保壁纸定时切换
- **开机自启**：`BootReceiver` 监听 `BOOT_COMPLETED`，恢复所有闹钟
- **图片权限**：通过 `ActivityResultContracts.GetContent` 选择图片，获取持久化 URI 读取权限

## 3. UI/UX 设计规范

### 3.1 主界面

```
┌──────────────────────────────────┐
│  Toolbar（主题色背景 + 设置入口）  │
├──────────────────────────────────┤
│  RecyclerView 任务卡片列表        │
│  ┌────────────────────────────┐  │
│  │ [52×92 缩略图]  08:30 每天  │  │
│  │                文件名       │  │
│  │                [桌面壁纸]   🗑│  │
│  └────────────────────────────┘  │
│                          [FAB +] │
└──────────────────────────────────┘
```

- 卡片：MaterialCardView，圆角 6dp，阴影 2dp
- 缩略图：竖版 16:9（52×92dp），`centerCrop`，位于卡片最左侧
- 文字信息：三行左对齐，居中区域
- 删除按钮：40dp 红色垃圾桶图标，卡片最右侧

### 3.2 新建任务流程

```
日子选择 → 时间选择 → 图片选择 → 壁纸类型选择 → 创建完成
```

### 3.3 对话框按钮规范

全部使用 `MaterialAlertDialogBuilder`（时间选择器为系统 `TimePickerDialog`）。按钮经 `styleDialogButtons()` 统一着色。

| 对话框 | 按钮布局（左→右） |
|--------|-------------------|
| 删除确认 | 删除 |
| 日子选择 | 确定 |
| 时间选择 | 上一步 \| 确定 |
| 壁纸类型 | 上一步 \| 确定 |
| 设置 | 确定 |

### 3.4 颜色体系

全部有色彩元素经 `desaturateColor()` 降低 50% 饱和度，保持柔和观感。

| 元素 | 色值 | 说明 |
|------|------|------|
| 确定按钮 | #4CAF50 ⊗50% | Material Green 500 |
| 上一步按钮 | #FF9800 ⊗50% | Material Orange 500 |
| 删除按钮 | #F44336 ⊗50% | Material Red 500 |
| 工具栏/FAB | 9 色主题色 ⊗50% | 用户可选 |
| Switch 开关 | #4CAF50 ⊗50% | 绿色开启态 |

## 4. 技术架构

### 4.1 文件结构

```
CWP/
├── app/
│   ├── build.gradle.kts          # 自动版本号递增
│   ├── version.properties        # 当前版本计数器
│   └── src/main/
│       ├── AndroidManifest.xml
│       ├── java/com/CWP/app/
│       │   ├── MainActivity.kt    # 主界面 + 全部业务逻辑
│       │   ├── WallpaperReceiver.kt  # 闹钟广播接收
│       │   └── BootReceiver.kt    # 开机广播接收
│       └── res/
│           ├── layout/
│           │   ├── activity_main.xml
│           │   ├── item_wallpaper_task.xml
│           │   └── dialog_settings.xml
│           ├── drawable/          # 药丸背景、图标等
│           └── values/            # 字符串、主题、颜色
```

### 4.2 数据模型

```kotlin
data class WallpaperTask(
    id: Long,           // System.currentTimeMillis()
    hour: Int,          // 0–23
    minute: Int,        // 0–59
    imageUri: String,   // content:// URI
    displayName: String,
    wallpaperFlag: Int, // FLAG_SYSTEM / FLAG_LOCK / 两者 or
    dayType: DayType
)
```

持久化格式（JSON Array，存于 `cwp_prefs` → `tasks_json`）：
```json
[{"id":1710000000000,"hour":8,"minute":30,"uri":"content://...","name":"bg.jpg","flag":1,"dayType":"EVERYDAY"}]
```

### 4.3 闹钟调度

- 首次调度：`MainActivity.scheduleAlarm()`
- 重复调度：`WallpaperReceiver.rescheduleAlarm()`
- 开机恢复：`BootReceiver.onReceive()`
- 仅一次类型：执行后 `WallpaperReceiver` 自动从存储中删除
- 闹钟类型：`AlarmManager.setAlarmClock()`（精确闹钟，系统可显示）

### 4.4 关键依赖

| 依赖 | 用途 |
|------|------|
| Material Components | MaterialCardView, MaterialButton, Chip, FAB, MaterialAlertDialogBuilder |
| AppCompat | AppCompatActivity, 夜间模式, 沉浸式 |
| RecyclerView | 任务卡片列表 |
| ConstraintLayout | 主界面布局 |
