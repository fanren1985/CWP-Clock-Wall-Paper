# CWP 壁纸定时切换 — 版本功能升级日志

> 基于开发对话记录整理，`VERSION_CODE` 仅作标记参考。

## VERSION_CODE 16 — 2026-05-26

### 🐛 Bug 修复

- **AlertDialog 类型不匹配**：`MaterialAlertDialogBuilder.show()` 返回 `androidx.appcompat.app.AlertDialog`，与 `styleDialogButtons()` 的参数 `android.app.AlertDialog` 类型冲突。修复方案：参数改为通用 `Dialog`，内部使用 `findViewById(android.R.id.button1/2/3)` 取按钮。
- **MaterialButton 按钮消失**：`MaterialButton` 直接设置 `background` 会破坏 MaterialShapeDrawable 渲染管线。修复为：MaterialButton 使用 `backgroundTintList` + `cornerRadius`；普通 Button 使用 `GradientDrawable`。
- **按钮顺序错误**：Android AlertDialog 标准排列为 `Neutral → Negative → Positive`（左→右），之前将 `Negative` 映射为"上一步"导致顺序颠倒。修正为 `Neutral="上一步"`、`Negative="取消"`、`Positive="确定"`。

### ✨ 功能优化

- **任务卡片重新设计**：
  - 缩略图缩小为原尺寸的 65%（52×92dp），从中央移至最左侧
  - 文字信息居中显示 → 改为三行左对齐
  - 删除按钮固定卡片最右侧
  - 删除图标 tint 从 XML 移至代码管理

- **"仅一次"说明文字**：日子选择器中追加 `（执行完毕自动删除）`

- **全局去除"取消"按钮**：删除确认、日子选择、壁纸类型选择、时间选择四个页面均移除取消按钮

- **时间选择器按钮间距**："上一步"与"确定"之间增加 48dp 间距

- **颜色体系统一**：
  - 新增 `desaturateColor()` 函数（HSV 空间 S×0.5），全局有色彩元素均降低 50% 饱和度
  - 按钮颜色统一为 Material Design 500 标准色：
    - 确定：#4CAF50（Green 500）
    - 上一步：#FF9800（Orange 500）
    - 取消/删除：#F44336（Red 500）
  - 主题色圆形按钮、Switch 开关、工具栏、FAB、状态栏同步去饱和处理

- **按钮横向间距收紧**：`padH` 从 28dp 缩减至 16dp，确保三按钮场景单行不换行
